环境准备

  • 安装 GIT

  • 安装 node.js:node,js download

    • 检查是否安装成功

  • 由于国内互联网环境，导致npm下载失败，可使用npm淘宝镜像cnpm*

  • 安装

搭建启动部署博客

  1. 新建一个放置博客文件的专属文件夹，在命令行下将路径切换到文件夹所在路径

  2. 博客初始化 命令行输入 hexo init 【mac os 输入 sudo hexo init】

  3. 启动 hexo s

  4. 新建一篇文章

    hexo n “我的第一篇博客文章.md”

  5. github 新建一个仓库 用户名.github.io

  6. 安装cnpm插件

  7. 修改_config.yml文件，在文件底部这样修改

  8. 部署到远端 hexo d

    博文用Markdown写，后缀名为.md，放在source/posts文件夹里。每次写完，hexo g -d生成部署

  9. 浏览器访问：870727741.github.io

更换blog主题

  1. 路径切换到blog文件夹下面

  2. 修改配置文件 blog/_config.yml
    _config.yml:theme: yilia

总结注意事项

  • git,github,repositories所有用户名/账户名一致一一致！
  • 每行命令失败先检查当前行是否敲错，根据错误提示检查敲过的命令是否生效
