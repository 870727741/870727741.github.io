搭建启动部署博客

  1. 新建一个放置博客文件的专属文件夹，在命令行下将路径切换到文件夹所在路径

  2. 博客初始化 命令行输入 hexo init 【mac os 输入 sudo hexo init】

  3. 启动 hexo s

  4. 新建一篇文章

    hexo n “我的第一篇博客文章.md”

  5. github 新建一个仓库 用户名.github.io jmp.github.io

  6. 安装cnpm插件

    博文用Markdown写，后缀名为.md，放在post文件夹里。每次写完，hexo g -d生成部署

    更换blog主题

  10. 修改配置文件 blog/_config.yml
    start _config.yml

总结注意事项

  • git,github,repositories所有用户名/账户名一致一一致！
  • 每行命令失败先检查当前行是否敲错，根据错误提示检查敲过的命令是否生效
